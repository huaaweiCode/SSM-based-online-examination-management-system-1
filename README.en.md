# 基于SSM的在线考试管理系统1

#### Description
基于SSM的在线考试管理系统1拥有三种角色

- 管理员：学生管理、班级管理、科目管理、试卷管理、自动生成试卷、待考管理、以往考试信息等
- 教师：学生班级科目的查询、试卷管理、考试安排等
- 用户：考试、登录注册、查询以往考试安排
